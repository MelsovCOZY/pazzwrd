# Contributing to pazzwrd

Thanks for your interest in contributing!

## Getting Started

```sh
git clone https://github.com/MelsovCOZY/pazzwrd.git
cd pazzwrd
bun install
```

## Development

```sh
bun src/cli.ts        # Run directly
bun test              # Run tests
bun run build         # Build for npm
```

## Making Changes

1. Fork the repo and create a branch from `main`
2. Write tests for your changes
3. Make sure all tests pass: `bun test`
4. Submit a pull request

## Reporting Bugs

Open an [issue](https://github.com/MelsovCOZY/pazzwrd/issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your OS and Node/Bun version

## Feature Requests

Open an [issue](https://github.com/MelsovCOZY/pazzwrd/issues) describing the feature and why it would be useful.
