# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in pazzwrd, please report it responsibly.

**Do not open a public issue.** Instead, email **yrnrmlsv@gmail.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours.

## Scope

pazzwrd generates passwords locally. It does not transmit data over the network. Security concerns would include:

- Weaknesses in randomness generation
- Predictable output patterns
- Dependencies with known vulnerabilities
